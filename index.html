Trabalho de WebGL de texturas
Trabalho final com AFrame

Para rodar ambos os trabalhos, utilizei o http-server do Node, instalado via: npm install -g http-server

Depois para rodar, vou na página dos arquivos em questão e executo: "http-server -p 3000 --add-cors"

Daí eu abro o navegador na porta escolhida e abro o navegador: http://localhost:3000/"nome do arquivo.html"


Obs: Sem o servidor o CORS estará bloqueado pelo navegador e não poderei usar texturas nos componentes
